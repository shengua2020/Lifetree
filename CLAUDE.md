# Life Tree Development Rules

This project follows strict design philosophy.

Core idea:

A tree grows with your life.

One sentence a day becomes one leaf.
365 leaves form one tree.
Multiple trees form a forest.


Design principles

• Apple-style minimal design
• Calm interface
• Natural colors
• Quiet animations
• No clutter
• No gamification


Interaction rules

• Records cannot be deleted
• Records cannot be edited
• The first leaf never falls
• Trees grow naturally with time
• Users discover the seed themselves
• Leaves appear slowly


Visual style

Inspired by Apple product design.

Interfaces must feel calm, quiet and natural.

Animations must be slow and subtle.


Technology stack

Frontend
Next.js

Graphics
SVG tree rendering

Animation
CSS animations

Backend
Simple API

Database
PostgreSQL
